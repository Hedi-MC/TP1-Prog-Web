Les Classes

Le Mage

  • Nombre de compétences de combat : 3
  • Nombre de compétences d'utilité : 2
  • Nombre de compétences d'amélioration : 1

Le mage est la classe la plus flexible, elle contient plusieurs habiletés utile pour plusieurs utilisations. Le mage n'a pas besoin d'autres classes pour l'aider à réussir, mais il est difficile à améliorer. Il possède aussi l'habileté d'enchantement, qui le rend essentiel à n'importe quelle classe.

Il ne se spécialise dans rien, mais est très utile seul.

Le Guerrier

  • Nombre de compétences de combat : 5
  • Nombre de compétences d'utilité : 1
  • Nombre de compétences d'amélioration : 0

Le Guerrier excèle dans tout ce qui concèrne le combat et la défense. Il contient toutes les compétences de combat qui ne consistent pas de la magie. De plus, il comprend l'habileté du forgeron, ce qui est utile pour toute classe.

Il se spécialise dans les compétences de combat, notamment le combat au corp à corp et les armes lourdes.

Le Voleur

  • Nombre de compétences de combat : 1
  • Nombre de compétences d'utilité : 4
  • Nombre de compétences d'amélioration : 1

La classe du voleur possède plusieurs habiletés qui lui permet de mieux se faufiler et éviter les enemies ainsi que de comettre des actes criminels, tout en ayant l'aptitude d'éviter ses répercussions.

Il se spécialise dans les compétences hors du combat, notamment la création de potions ainsi que le vol.
